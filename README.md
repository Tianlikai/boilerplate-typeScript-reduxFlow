# boilerplate

这是一个快速生成含有 typescript 语言环境，自定义 redux 和 redux-saga 数据流的 React 种子项目

## 拉取项目

通过一下命令拉取项目，然后根据提示进行初始化项目

```bash
    npm i yo generator-tboilterplate -g
    yo tboilterplate
```
